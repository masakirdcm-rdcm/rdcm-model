# RDCM Model

Minimal implementation of the Recursive Differentiation–Convergence Model (RDCM).

## Overview
This repository contains simulation code for RDCM, a minimal computational model describing how stable directional structure emerges from persistent variability.

## Files
- `timeseries.py` : generates time series (Figure 2)
- `phase.py` : generates phase diagram (Figure 3)

## Usage
```bash
python timeseries.py
python phase.py
